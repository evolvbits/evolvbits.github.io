Credits

EvolvBits uses several open-source technologies and resources that help power the website and development workflow. We acknowledge and thank the creators and maintainers of these projects.

Open Source Technologies

The EvolvBits website is built using the following open-source technologies.

Static Site Generator

  • Jekyll — Static site generator used to build and manage the website.

Frontend Framework

  • Bootstrap — CSS framework used for layout, components, and responsive design.

Icons

  • Bootstrap Icons — Icon library used throughout the interface.

Styling

  • Sass (Syntactically Awesome Style Sheets) — CSS preprocessor used to structure and maintain stylesheets.

Development Runtime

  • Node.js — Used in the development environment to manage tooling and build processes.
