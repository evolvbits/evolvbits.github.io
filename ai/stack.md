Stack técnica obrigatória do projeto:

- Jekyll
- Bootstrap 5.3 (sistema principal de layout e utilities)
- Dart Sass moderno (OBRIGATÓRIO usar @use e @forward, PROIBIDO @import)
- HTML semântico

Regras importantes:

Bootstrap 5.3 é responsável por:
- Containers
- Grid system
- Flex utilities
- Spacing utilities
- Responsividade

É PROIBIDO recriar qualquer funcionalidade que o Bootstrap 5.3 já fornece.

O SCSS do projeto existe apenas para:
- Sistema de cores
- Tipografia
- Tema visual
- Componentes específicos do projeto
