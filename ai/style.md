Regras de estilo:

- Usar HTML semântico
- Usar a metodologia BEM
- Prioridade para dispositivos móveis
- Variáveis ​​SCSS para cores
- Sem estilos inline

Sass Rules (OBRIGATÓRIO)

Este projeto usa Dart Sass moderno.

É PROIBIDO usar:
- @import
- funções Sass legadas como if()

É OBRIGATÓRIO usar:
- @use
- @forward
- módulos Sass
- sintaxe moderna compatível com Dart Sass 3+

Toda organização de SCSS deve seguir o sistema de módulos.
